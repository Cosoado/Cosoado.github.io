Privacy Policy

App Name OmakaseMaster (Sushi App)

Provider Hideaki Murasawa

Category Lifestyle・Education

1. Basic Policy

OmakaseMaster (hereinafter referred to as "this App") places the highest priority on user privacy. This App provides sushi-related learning and information services, and we take every possible measure to protect users' personal information.

2. Information We Collect

This App operates based on the following principles:

  • No external transmission of personal information: Personal information entered by users (learning history, settings, etc.) is never transmitted to external servers
  • Local storage only: All input data is stored only within the user's device
  • Automatic deletion: All stored data will be completely deleted when the app is uninstalled

3. Purpose of Data Use

Collected data is used only for the following purposes:

  • Local storage of learning history
  • App functionality improvement
  • User experience enhancement

4. Sharing with Third Parties

This App does not share user information with third parties except for the following:

  • Google AdMob: Only technical information such as device identifiers for advertising delivery
  • Legal obligations: When there is a disclosure request based on laws and regulations

5. About Advertising

This App uses Google AdMob to display advertisements:

  • Displays banner ads and reward ads
  • Personalized ads are disabled
  • Uses only minimal technical information for ad display
  • We are not responsible for the content of advertisements

6. Data Storage Period

  • Learning data: Stored locally until deleted by user
  • Settings data: Stored locally until deleted by user
  • App deletion: All data is automatically deleted

7. Security

This App implements the following security measures:

  • Encrypted data storage
  • Use of secure communication protocols (HTTPS)
  • Regular security audits
  • Application of latest security updates

8. User Rights

Users have the following rights:

  • Viewing stored data
  • Deleting stored data
  • Discontinuing app use
  • Inquiring about this policy

9. Use by Minors

This App is not intended for use by persons under 13 years of age. If persons under 13 years of age use this App, please obtain parental consent before use.

10. Integration with Third-Party Services

This App uses the following external services:

  • Google AdMob: Advertising delivery
  • Firebase: App version management only

Please also review the privacy policies of these services.

11. Policy Changes

This Privacy Policy may be updated as necessary. Important changes will be notified within the app. The latest version can be found on this page.

12. Governing Law

This Privacy Policy is governed by Japanese law, and Japanese courts shall have jurisdiction.

Contact

For inquiries regarding this Privacy Policy, please contact via App Store Connect.

App Provider Hideaki Murasawa

プライバシーポリシー

アプリ名 OmakaseMaster（寿司アプリ）

提供者 Hideaki Murasawa

カテゴリ ライフスタイル・教育

1. 基本方針

OmakaseMaster（以下「本アプリ」）は、ユーザーのプライバシーを最重要視しています。本アプリは、寿司に関する学習・情報提供サービスを提供しており、ユーザーの個人情報の保護に万全を期しています。

2. 収集する情報

本アプリは以下の原則に基づいて運営されています：

  • 個人情報の外部送信なし: ユーザーが入力した個人情報（学習履歴、設定等）は一切外部サーバーに送信されません
  • ローカル保存のみ: 全ての入力データはユーザーの端末内にのみ保存されます
  • 自動削除: アプリを削除すると、保存された全てのデータが完全に削除されます

3. データの利用目的

収集したデータは以下の目的でのみ利用します：

  • 学習履歴の端末内保存
  • アプリの機能向上
  • ユーザー体験の向上

4. 第三者との共有

本アプリは以下を除き、ユーザーの情報を第三者と共有することはありません：

  • Google AdMob: 広告配信のため、デバイス識別子等の技術的情報のみ
  • 法的義務: 法令に基づく開示要求がある場合

5. 広告について

本アプリではGoogle AdMobを使用して広告を表示します：

  • バナー広告とリワード広告を表示
  • パーソナライズされた広告は無効に設定
  • 広告表示のため最小限の技術的情報のみ使用
  • 広告の内容について当方は責任を負いません

6. データの保存期間

  • 学習データ: ユーザーが削除するまで端末内に保存
  • 設定データ: ユーザーが削除するまで端末内に保存
  • アプリ削除時: 全てのデータが自動的に削除

7. セキュリティ

本アプリは以下のセキュリティ対策を実施しています：

  • データの暗号化保存
  • 安全な通信プロトコル（HTTPS）の使用
  • 定期的なセキュリティ監査
  • 最新のセキュリティアップデートの適用

8. ユーザーの権利

ユーザーは以下の権利を有します：

  • 保存されたデータの確認
  • 保存されたデータの削除
  • アプリの利用停止
  • 本ポリシーに関する問い合わせ

9. 未成年者の利用

本アプリは13歳未満の方の利用を想定していません。13歳未満の方が利用される場合は、保護者の同意を得てからご利用ください。

10. 他社サービスとの連携

本アプリは以下の外部サービスを利用しています：

  • Google AdMob: 広告配信
  • Firebase: アプリバージョン管理のみ

これらのサービスのプライバシーポリシーもご確認ください。

11. ポリシーの変更

本プライバシーポリシーは必要に応じて更新される場合があります。重要な変更については、アプリ内で通知いたします。最新版は本ページでご確認いただけます。

12. 準拠法

本プライバシーポリシーは日本法に準拠し、日本の裁判所を管轄裁判所とします。

お問い合わせ

本プライバシーポリシーに関するお問い合わせは、App Store Connect経由でお願いいたします。

隐私政策

应用名称 OmakaseMaster（寿司应用）

提供者 Hideaki Murasawa

类别 生活方式・教育

1. 基本方针

OmakaseMaster（以下简称"本应用"）将用户隐私视为最重要的事项。本应用提供寿司相关的学习和信息服务，我们在保护用户个人信息方面做到万无一失。

2. 收集的信息

本应用基于以下原则运营：

  • 不向外部发送个人信息: 用户输入的个人信息（学习历史、设置等）绝不会发送到外部服务器
  • 仅本地存储: 所有输入数据仅存储在用户设备内
  • 自动删除: 删除应用时，所有存储的数据都会被完全删除

3. 数据使用目的

收集的数据仅用于以下目的：

  • 学习历史的设备内存储
  • 应用功能改进
  • 用户体验提升

4. 与第三方共享

除以下情况外，本应用不会与第三方共享用户信息：

  • Google AdMob: 仅为广告投放提供设备标识符等技术信息
  • 法律义务: 基于法律法规有披露要求时

5. 关于广告

本应用使用Google AdMob显示广告：

  • 显示横幅广告和奖励广告
  • 个性化广告设置为禁用
  • 仅使用广告显示所需的最少技术信息
  • 我们对广告内容不承担责任

6. 数据存储期间

  • 学习数据: 存储在设备内直到用户删除
  • 设置数据: 存储在设备内直到用户删除
  • 应用删除时: 所有数据自动删除

7. 安全性

本应用实施以下安全措施：

  • 数据加密存储
  • 使用安全通信协议（HTTPS）
  • 定期安全审计
  • 应用最新安全更新

8. 用户权利

用户拥有以下权利：

  • 确认存储的数据
  • 删除存储的数据
  • 停止使用应用
  • 就本政策进行咨询

9. 未成年人使用

本应用不适用于13岁以下人员使用。如果13岁以下人员使用，请在获得家长同意后使用。

10. 与其他公司服务的协作

本应用使用以下外部服务：

  • Google AdMob: 广告投放
  • Firebase: 仅用于应用版本管理

请也确认这些服务的隐私政策。

11. 政策变更

本隐私政策可能会根据需要进行更新。重要变更将在应用内通知。最新版本可在本页面确认。

12. 适用法律

本隐私政策适用日本法，以日本法院为管辖法院。

联系方式

有关本隐私政策的咨询，请通过App Store Connect联系。

OmakaseMaster（이하 "본 앱"）은 사용자의 프라이버시를 최우선으로 고려합니다. 본 앱은 스시에 관한 학습・정보 제공 서비스를 제공하며, 사용자의 개인정보 보호에 만전을 기하고 있습니다.

본 앱은 다음 원칙에 기반하여 운영됩니다:

  • 개인정보의 외부 전송 없음: 사용자가 입력한 개인정보（학습 이력, 설정 등）는 일체 외부 서버로 전송되지 않습니다

본 앱은 다음을 제외하고 사용자의 정보를 제3자와 공유하지 않습니다:

  • Google AdMob: 광고 배신을 위해 디바이스 식별자 등의 기술적 정보만
