OshiVista プライバシーポリシー

最終更新日: 2025年11月9日

トップページ 利用規約

OshiVista（以下「本アプリ」といいます）の提供者（以下「当方」といいます）は、本アプリをご利用になる方（以下「ユーザー」といいます）のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシー（以下「本ポリシー」といいます）は、本アプリがどのような情報を収集し、どのように利用・管理するかについて説明するものです。

第1条（個人情報の定義）

本ポリシーにおいて「個人情報」とは、個人情報の保護に関する法律（以下「個人情報保護法」といいます）第2条第1項に定義される「個人情報」を意味するものとします。

第2条（プライバシーポリシーの適用範囲）

  1. 本ポリシーは、本アプリが提供するサービスにおいてのみ適用されます。
  2. 本アプリが利用する外部サービス（OpenAI、Perplexity、Firebase等）については、各サービスのプライバシーポリシーが適用されます。ユーザーは、本アプリの利用に際して、これらの外部サービスのプライバシーポリシーも確認することを推奨します。

第3条（収集する情報）

3.1 基本的な情報収集方針

本アプリは、原則としてユーザーの個人情報を収集しません。本アプリは、ユーザー登録やログイン機能を持たず、ユーザーが入力したデータは全てユーザーの端末内にローカル保存されます。

3.2 ユーザーが入力するデータ（ローカル保存）

以下の情報は、ユーザーの端末内にのみ保存され、当方のサーバーには送信されません：

  • 推し対象の名前、ニックネーム、プロフィール情報
  • イベントスケジュール情報（イベント名、日時、場所等）
  • 写真、画像データ
  • メモ、コメント等のテキストデータ
  • アプリ設定情報（言語設定、通知設定等）

3.4 自動的に収集される情報

本アプリは、サービスの改善とセキュリティ向上のため、以下の情報を自動的に収集する場合があります：

  • 端末情報（OS種別、OSバージョン、端末モデル、画面解像度等）
  • アプリの利用状況（起動回数、機能の利用状況、クラッシュ情報等）
  • ネットワーク接続情報（IPアドレス、通信キャリア情報等）

これらの情報は、個人を特定できない形式で収集・処理されます。

第4条（情報の利用目的）

当方は、収集した情報を以下の目的で利用します：

  1. 本アプリのサービス提供、維持、保護および改善のため
  2. ユーザーからのお問い合わせに対応するため
  3. 本アプリの新機能、更新情報、キャンペーン等のご案内のため
  4. 本アプリの利用規約に違反する行為に対応するため
  5. 本アプリの利用状況を分析し、サービスの改善に役立てるため
  6. セキュリティの維持および不正行為の防止のため
  7. 前各号に付随する目的のため

第5条（第三者への情報提供）

5.1 原則

当方は、ユーザーの同意なく、収集した情報を第三者に提供することはありません。ただし、以下の場合を除きます：

5.2 提供する場合

  1. 法令に基づく場合
  2. 人の生命、身体または財産の保護のために必要がある場合であって、ユーザーの同意を得ることが困難である場合
  3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、ユーザーの同意を得ることが困難である場合
  4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、ユーザーの同意を得ることにより当該事務の遂行に支障を及ぼすおそれがある場合
  5. 外部サービス連携時に、サービス提供のために必要な範囲で情報を送信する場合（第3条3.3に記載の通り）

5.3 外部サービスとの連携

本アプリは、以下の外部サービスと連携しています。これらのサービスに対しては、サービス提供に必要な範囲で情報が送信されます：

  • OpenAI: AI機能の提供のため、ユーザーが入力したテキスト情報や検索クエリが送信されます。
  • Perplexity: 情報検索機能の提供のため、推し対象の名前や検索クエリが送信されます。
  • Firebase: アプリの設定管理、分析、クラッシュレポートのため、匿名化された利用統計情報が送信されます。

第6条（データの保存場所と期間）

6.1 データの保存場所

ユーザーデータは、原則としてユーザーの端末内にローカル保存されます。当方のサーバーには保存されません。

  • 端末内ローカル保存： 推し対象情報、イベント情報、写真、メモ等、ユーザーが入力した全てのデータ
  • 外部サービス： AI機能や情報検索機能を利用した際の一時的なデータは、各外部サービスのサーバーで処理されますが、その保存期間は各サービスのポリシーに従います

6.2 データの保存期間

  • ローカルデータ： ユーザーがアプリをアンインストールするまで、または手動で削除するまで保存されます
  • 分析データ： 収集から最大26ヶ月間保存されます（Firebaseのデフォルト設定）
  • 外部サービスでの処理データ： 各サービスのデータ保持ポリシーに従います

第7条（データのバックアップとエクスポート）

重要な注意事項

本アプリのデータは端末内にローカル保存されるため、以下の場合にデータが失われる可能性があります：

  • アプリのアンインストール
  • 端末の故障、紛失、盗難
  • OSのアップデートや初期化
  • 端末の買い替え

当方は、ユーザーデータのバックアップを提供しておりません。ユーザーは、重要なデータについては、定期的にスクリーンショットの保存やデータのエクスポート（将来実装予定）等、自己の責任でバックアップを行うことを強く推奨します。

第8条（セキュリティ）

  1. 当方は、収集した情報の漏洩、滅失または毀損の防止その他の情報の安全管理のために必要かつ適切な措置を講じます。
  2. ただし、本アプリはユーザーの端末内にデータを保存するため、端末のセキュリティ対策はユーザー自身の責任となります。
  3. ユーザーは、以下のセキュリティ対策を実施することを推奨します：
    • 端末のパスコードロックまたは生体認証の設定
    • OSの最新バージョンへのアップデート
    • 信頼できないWi-Fiネットワークでの利用を避ける
    • 端末の紛失・盗難時の遠隔ロック機能の有効化

第9条（ユーザーの権利）

ユーザーは、以下の権利を有します：

  1. アクセス権： 本アプリ内で自身のデータを閲覧することができます
  2. 訂正権： 本アプリ内で自身のデータを編集・訂正することができます
  3. 削除権： 本アプリ内で自身のデータを削除することができます。また、アプリをアンインストールすることで、端末内の全てのデータを削除することができます
  4. データポータビリティ権： 将来的に、自身のデータをエクスポートする機能を提供する予定です
  5. 利用停止権： いつでも本アプリの利用を停止し、アンインストールすることができます

第10条（Cookie及びこれに類する技術の利用）

  1. 本アプリは、モバイルアプリケーションであるため、Webブラウザで使用されるCookieは使用しません。
  2. ただし、Firebase Analyticsなど一部の外部サービスでは、類似の技術を使用してユーザーの行動を追跡する場合があります。これらの技術は、個人を特定できない形式で利用されます。
  3. ユーザーは、端末の設定により、これらの追跡機能を無効にすることができます。ただし、一部機能が正常に動作しなくなる可能性があります。

第11条（子供のプライバシー）

  1. 本アプリは、13歳未満の子供を対象としたサービスではありません。
  2. 当方は、意図的に13歳未満の子供から個人情報を収集することはありません。
  3. 13歳未満の子供が本アプリを利用する場合は、保護者の同意と監督の下で利用してください。
  4. 万一、13歳未満の子供から個人情報を収集したことが判明した場合、当方は速やかに当該情報を削除します。

第12条（広告配信について）

現在、本アプリでは広告配信を行っておりません。

将来的に広告配信を開始する場合は、本ポリシーを更新し、ユーザーに通知します。

第13条（分析ツールの使用）

本アプリは、サービスの改善のため、以下の分析ツールを使用しています：

Firebase Analytics

  • 提供元: Google LLC
  • 収集される情報: 匿名の利用統計情報、アプリの動作状況、クラッシュレポート
  • 利用目的: アプリの品質向上、バグの修正、ユーザー体験の改善
  • プライバシーポリシー: Firebase プライバシーとセキュリティ

これらの分析ツールは、個人を特定できない形式でデータを収集・処理します。

第14条（プライバシーポリシーの変更）

  1. 当方は、法令の変更、サービス内容の変更、その他の事由により、本ポリシーを変更することがあります。
  2. 本ポリシーを変更する場合、変更後のプライバシーポリシーの効力発生日の相当期間前に、本アプリ内の適切な場所への表示その他の適切な方法により周知します。
  3. 重要な変更がある場合は、本アプリ内で目立つ通知を行います。
  4. 変更後のプライバシーポリシーは、効力発生日から効力を生じるものとします。
  5. ユーザーは、プライバシーポリシーの変更後も本アプリの利用を継続することにより、変更後のプライバシーポリシーに同意したものとみなされます。

第15条（個人情報の開示・訂正・削除）

  1. 本アプリは、ユーザーの端末内にデータをローカル保存しているため、ユーザーは本アプリ内でいつでも自身のデータを閲覧、編集、削除することができます。
  2. 外部サービスに送信されたデータについて開示、訂正、削除を希望される場合は、各外部サービスのプライバシーポリシーに従って手続きを行ってください。
  3. 当方が保有する分析データ等について開示、訂正、削除を希望される場合は、第16条に記載のお問い合わせ先までご連絡ください。ただし、匿名化されたデータについては、個人を特定できないため、対応できない場合があります。
