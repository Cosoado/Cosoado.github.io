OshiVista 隐私政策

最后更新日期：2025年11月9日

首页 使用条款

OshiVista（以下简称"本应用"）的提供者（以下简称"我方"）尊重使用本应用的用户（以下简称"用户"）的隐私，致力于保护个人信息。本隐私政策（以下简称"本政策"）说明本应用收集何种信息以及如何使用和管理这些信息。

第1条（个人信息的定义）

本政策中的"个人信息"是指《个人信息保护法》第2条第1项中定义的"个人信息"。

第2条（隐私政策的适用范围）

  1. 本政策仅适用于本应用提供的服务。
  2. 对于本应用使用的外部服务（OpenAI、Perplexity、Firebase等），适用各服务的隐私政策。建议用户在使用本应用时，也确认这些外部服务的隐私政策。

第3条（收集的信息）

3.1 基本信息收集方针

本应用原则上不收集用户的个人信息。本应用不具备用户注册或登录功能，用户输入的数据全部保存在用户设备的本地存储中。

3.2 用户输入的数据（本地保存）

以下信息仅保存在用户设备中，不会发送至我方服务器：

  • 爱豆的姓名、昵称、个人资料信息
  • 活动日程信息（活动名称、日期、地点等）
  • 照片、图像数据
  • 备忘录、评论等文本数据
  • 应用设置信息（语言设置、通知设置等）

3.4 自动收集的信息

为了改进服务和提高安全性，本应用可能会自动收集以下信息：

  • 设备信息（操作系统类型、操作系统版本、设备型号、屏幕分辨率等）
  • 应用使用情况（启动次数、功能使用情况、崩溃信息等）
  • 网络连接信息（IP地址、运营商信息等）

这些信息以无法识别个人的形式进行收集和处理。

第4条（信息的使用目的）

我方将收集的信息用于以下目的：

  1. 提供、维护、保护和改进本应用的服务
  2. 回应用户的咨询
  3. 通知本应用的新功能、更新信息、活动等
  4. 应对违反本应用使用条款的行为
  5. 分析本应用的使用情况，以改进服务
  6. 维护安全和防止欺诈行为
  7. 与前各项相关的目的

第5条（向第三方提供信息）

5.1 原则

我方不会在未经用户同意的情况下向第三方提供收集的信息。但以下情况除外：

5.2 提供的情况

  1. 根据法律规定时
  2. 为保护人的生命、身体或财产而有必要且难以获得用户同意时
  3. 为改善公共卫生或促进儿童健康成长而特别需要且难以获得用户同意时
  4. 需要配合国家机关、地方公共团体或其受托者履行法律规定的事务，且获得用户同意可能妨碍该事务履行时
  5. 连接外部服务时，在提供服务所需的范围内发送信息（如第3条3.3所述）

5.3 与外部服务的连接

本应用与以下外部服务连接。在提供服务所需的范围内，会向这些服务发送信息：

  • OpenAI：为提供AI功能，会发送用户输入的文本信息和搜索查询。
  • Perplexity：为提供信息搜索功能，会发送爱豆的姓名和搜索查询。
  • Firebase：为进行应用配置管理、分析和崩溃报告，会发送匿名化的使用统计信息。

第6条（数据的保存位置和期限）

6.1 数据的保存位置

用户数据原则上保存在用户设备的本地存储中。不会保存在我方服务器上。

  • 设备本地存储：用户输入的所有数据，包括爱豆信息、活动信息、照片、备忘录等
  • 外部服务：使用AI功能或信息搜索功能时的临时数据会在各外部服务的服务器上处理，但其保存期限遵循各服务的政策

6.2 数据的保存期限

  • 本地数据：保存至用户卸载应用或手动删除为止
  • 分析数据：自收集起最多保存26个月（Firebase的默认设置）
  • 外部服务处理的数据：遵循各服务的数据保留政策

第7条（数据的备份和导出）

重要注意事项

由于本应用的数据保存在设备本地存储中，在以下情况下可能会丢失数据：

  • 卸载应用
  • 设备故障、丢失、被盗
  • 操作系统更新或初始化
  • 更换设备

我方不提供用户数据的备份服务。强烈建议用户自行负责定期备份重要数据，如保存截图或导出数据（计划在未来实现）。

第8条（安全性）

  1. 我方采取必要且适当的措施，防止收集信息的泄露、丢失或损坏，以及其他信息的安全管理。
  2. 但是，由于本应用将数据保存在用户设备上，设备的安全措施由用户自行负责。
  3. 建议用户实施以下安全措施：
    • 设置设备密码锁或生物识别认证
    • 将操作系统更新至最新版本
    • 避免在不可信的Wi-Fi网络上使用
    • 启用设备丢失或被盗时的远程锁定功能

第9条（用户的权利）

用户拥有以下权利：

  1. 访问权：可以在本应用内查看自己的数据
  2. 更正权：可以在本应用内编辑、更正自己的数据
  3. 删除权：可以在本应用内删除自己的数据。此外，通过卸载应用可以删除设备内的所有数据
  4. 数据可携带权：计划在未来提供导出自己数据的功能
  5. 停止使用权：可以随时停止使用本应用并卸载

第10条（Cookie及类似技术的使用）

  1. 本应用是移动应用程序，不使用网页浏览器中使用的Cookie。
  2. 但是，Firebase Analytics等部分外部服务可能使用类似技术来跟踪用户行为。这些技术以无法识别个人的形式使用。
  3. 用户可以通过设备设置禁用这些跟踪功能。但是，部分功能可能无法正常运行。

第11条（儿童隐私）

  1. 本应用不是面向13岁以下儿童的服务。
  2. 我方不会故意收集13岁以下儿童的个人信息。
  3. 如果13岁以下儿童使用本应用，请在监护人的同意和监督下使用。
  4. 如果发现收集了13岁以下儿童的个人信息，我方将立即删除该信息。

第12条（关于广告投放）

目前本应用不进行广告投放。

如果将来开始投放广告，我方将更新本政策并通知用户。

第13条（分析工具的使用）

为了改进服务，本应用使用以下分析工具：

Firebase Analytics

  • 提供商：Google LLC
  • 收集的信息：匿名使用统计信息、应用运行状况、崩溃报告
  • 使用目的：提高应用质量、修复错误、改善用户体验
  • 隐私政策：Firebase 隐私与安全

这些分析工具以无法识别个人的形式收集和处理数据。

第14条（隐私政策的变更）

  1. 我方可能因法律变更、服务内容变更或其他原因而变更本政策。
  2. 变更本政策时，我方将在变更后隐私政策生效日期的合理期限前，通过在本应用内适当位置显示或其他适当方式进行通知。
  3. 如有重大变更，我方将在本应用内进行显著通知。
  4. 变更后的隐私政策自生效日起生效。
  5. 用户在隐私政策变更后继续使用本应用，即视为同意变更后的隐私政策。

第15条（个人信息的披露、更正、删除）

  1. 由于本应用将数据保存在用户设备的本地存储中，用户可以随时在本应用内查看、编辑、删除自己的数据。
  2. 如果希望披露、更正、删除发送至外部服务的数据，请按照各外部服务的隐私政策进行操作。
  3. 如果希望披露、更正、删除我方持有的分析数据等，请通过第16条记载的联系方式与我方联系。但是，对于匿名化的数据，由于无法识别个人，可能无法响应。

第17条（准据法和管辖法院）

  1. 本政策的解释适用日本法律。
  2. 如果就本政策产生纠纷，以我方所在地的管辖法院为专属合意管辖法院。
